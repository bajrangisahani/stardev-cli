# ✨ Why STARDEV CLI?

Developers spend hours on repetitive tasks.

STARDEV CLI automates them.

Instead of using multiple tools for documentation, GitHub, deployment, screenshots, security, releases, and AI content generation, STARDEV brings everything together in a single modular CLI built with TypeScript.

# 📦 Installation

```bash
git clone https://github.com/bajrangisahani/stardev-cli.git

cd stardev-cli

npm install

npm run build

npm link
```

Run:

```bash
stardev --help
```

---

# 🚀 Quick Start

Initialize a new project

```bash
stardev init
```

Generate README

```bash
stardev readme
```

Publish to GitHub

```bash
stardev github
```

Deploy

```bash
stardev deploy
```

Health Check

```bash
stardev health
```

Security Scan

```bash
stardev security
```

# ⚙ Configuration

Store local encrypted configuration.

```bash
stardev config set

stardev config path
```

Supports:

- GitHub Token
- Vercel Token
- Author
- Company
- Portfolio Path
- Git Username
- Git Email
- Default License
